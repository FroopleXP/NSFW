  SwapMouseButtons
  An easy way to share a computer between left-handed and right-handed persons




Simple and Quick

 If you are left-handed and the computer is set up for right-handed use, press Ctrl-F12 to swap the mouse buttons. Each time you switch, an icon is displayed to indicate the main mouse button.		      



No Fumbling, No Frustration

 Just press one key combination to swap without having to fumble before switching. You do not have to use the mouse with the wrong hand until you switch.		      

Why Use This Program?

 The Windows control panel provides one way to swap the mouse's button assignments, but it requires too many steps. If a computer is shared, this program provides a simple solution.

       
Left-Handed Pointers

 When the mouse buttons are swapped, the mouse pointers are also automatically changed to left-handed pointers.
